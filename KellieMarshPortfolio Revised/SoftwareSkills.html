Experience from Software Developer Essentials Course

This 12 week intensive course began by introducing us to front end development using JavaScript, HTML and CSS building our own Portfolio websites. Next, working as part of a team of 6 developers, using pair programming we built the front end of a Bookswap website using React and CSS. At this stage we used a pre-built API for the back-end. We used Git Hub to aid collaboration.

We then moved on to back-end development using PHP , MySQL and Sequel Ace building a collection website. Creating our own databases in Sequel Ace, using MySQL to access and manipulate data from it. The website was built using PHP and HTML, styled in CSS. The website was built using object-oriented programming and following best practices for maintainability, such as writing unit tests.

The final project was to create our own API to replace the pre-built API attached to our Bookswap website. The API was built using PHP , Laravel , Sequel Ace and Postman.
